## Capture Tips

- Use PNG format.
- Recommended desktop resolution: 1920x1080.
- For the mobile screenshot, use browser device emulation around 390x844.
- Keep browser zoom at 100% if possible.
- Remove sensitive personal data from screenshots before committing.

## Suggested Capture Order

1. Capture the live website hero and highlights sections.
2. Capture a successful GitHub Actions run.
3. Capture Pages deployment environment details.
4. Capture the repository structure and key files.
5. Capture LinkedIn Featured section after profile update.
6. Capture the slide deck cover after the presentation is ready.
